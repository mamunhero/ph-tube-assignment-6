What is var, let, const?
Var

Variables declared with var are function-scoped. This means they are only accessible within the function in which they are defined. If declared outside any function, they become global variables and are accessible throughout the entire program.

Let

The let keyword is used to declare variables in JavaScript with block-level scope. Block scope means that a variable declared with let is only accessible within the block (enclosed by curly braces) where it's defined. This behavior provides more predictable and localized variable access compared to the older var keyword.

Const

The const keyword is used to declare variables in JavaScript with block-level scope, just like let. However, const has some unique characteristics related to immutability and reassignment.

Tell us the use cases of Null and Undefined?
Null

Null is a primitive value in JavaScript that represents the intentional absence of any object value or a variable having no assigned value.

undefined

Undefined is another primitive value in JavaScript. It represents a variable that has been declared but has not been assigned a value, or it can be the value returned from a function that doesn't explicitly return anything.

What do you mean by REST API?
REST API

REST API stands for "Representational State Transfer Application Programming Interface." It is an architectural style for designing networked applications, and it is based on a set of principles and constraints that help create scalable, stateless, and well-structured web services. REST is not a standard, but rather a set of guidelines and best practices commonly used in web development.
